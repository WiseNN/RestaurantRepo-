# RestaurantRepo-

This repo has been made to show individuals the structure of this React.js project. Please note that this is an unfinished restuarant website. 

To run this, clone the repo, install mongodb globally and run: `mongod` from any terminal. Then go into the parent directory, and run: `npm start`. Report to URL: `localhost:3000` . The components that have functionality are: quickMenu and Menu.
